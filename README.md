# Воппер Ниндзя

Интерактивный курс по новому продукту (соус «Чёрный Ниндзя») и стандартам
сборки и упаковки Воппера. Собран на дизайн-системе KU — чистый HTML/CSS/JS,
без сборки, SCORM 1.2. Открывается двойным кликом или на любом статик-хостинге.

## Что внутри

- **Глава 1 — Соус «Чёрный Ниндзя»**: видео + выжимка по работе с соусом.
- **Глава 2 — Тест по соусу**: заглушка (наполняется позже).
- **Глава 3 — Стандарты**: две игры —
  - «Собери Воппер» — перетаскивание ингредиентов в правильном порядке
    (раунд на порядок и раунд на время);
  - «Упаковка» — поворот оберточной бумаги + позиционирование Воппера
    к отметке свайпом.
- **Результаты**: обратная связь по играм; при провале нужное видео-разбор
  становится обязательным, при успехе — по желанию.

## Локальный запуск

```bash
python3 -m http.server 8792
# открыть http://localhost:8792
```

Локальный http-сервер нужен, чтобы корректно резолвились `@import` и шрифты.

## Права на ассеты

Шрифт Flame, логотип и оберточная бумага Burger King — товарные знаки и
материалы Burger King. Все права принадлежат правообладателю.
